# ⚡ Electricity Meter Dashboard (ApexCharts + React)

## 📌 Overview
This project is a web-based dashboard to visualize electricity meter data (M1, M2, M3, M4, Master) from a CSV file. It supports interactive charts, alerts, and configuration options.

---

## 🚀 Features
- 📊 Line and Stacked Bar Charts (ApexCharts)
- ⏱️ Time range filtering
- 🧠 Tooltip with meter values
- 🚨 Alerts:
  - High consumption (>1000W)
  - Leakage detection (>300W)
- 🎛️ Toggle alerts ON/OFF
- 📍 Alert markers on graph

---

## 🛠️ Tech Stack
- React (Functional Components + Hooks)
- Typescript
- ApexCharts
- Day.js
